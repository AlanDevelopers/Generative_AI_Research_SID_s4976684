What does git clone do ; what does github fork do. Differences between git clone and github fork

Git clone downloads a repository from GitHub to my local computer, so I can open the files, edit them, and work on them locally. GitHub fork creates my own copy of someone else’s repository under my GitHub account, so I can make changes without affecting the original project.

The main difference is that clone is mainly for getting the project onto my computer, while fork is for making a separate copy on GitHub. In a group project, I would usually fork the original repository first, then clone my forked version to my computer and work from there.
